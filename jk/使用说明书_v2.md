## 1. 系统简介

### 1.1 系统概述

本系统是由**石家庄铁道大学**开发的健康养生旅游模式可视化选择平台。系统通过数据化分析，帮助用户根据自身兴趣偏好，智能匹配最适合的康养旅游模式。

### 1.2 主要功能

- **四种康养模式**：文化+、生态+、休闲+、医疗+健康养生
- **智能匹配算法**：基于余弦相似度计算用户与模式的匹配程度
- **可视化图表**：折线图、饼状图、散点图多维度展示
- **AI智能助手**：小旅助手随时解答用户疑问
- **数据导出**：支持Excel和Word格式报告导出

### 1.3 技术架构

- 前端：HTML5 + CSS3 + JavaScript
- 图表库：Chart.js
- 数据处理：SheetJS (Excel导出)
- 文档生成：docx.js (Word导出)
- AI接口：DeepSeek API

### 7.2 操作步骤

#### 第一步：输入兴趣值

在四个输入框中填入0-10的兴趣评分：

| 维度 | 说明 | 建议分数 |
|------|------|----------|
| 文化兴趣 | 对历史、传统文化、非遗体验的兴趣 | 0-10 |
| 生态兴趣 | 对自然环境、森林、湿地等生态游的兴趣 | 0-10 |
| 休闲兴趣 | 对度假、SPA、美食等休闲活动的兴趣 | 0-10 |
| 医疗兴趣 | 对健康体检、中医养生、康复疗养的兴趣 | 0-10 |

#### 第二步：点击计算

点击"🧮 快速计算"按钮，系统将：
1. 读取您输入的兴趣值
2. 与四种模式特征向量计算余弦相似度
3. 找出匹配度最高的康养模式
4. 展示详细推荐结果

计算完成后，页面显示：

### 7.3 结果解读

- **余弦相似度**：数值越接近1，匹配度越高
- **匹配度**：系统推荐的置信度指标
- **推荐理由**：根据您的兴趣特征给出的解释
- **消费档次**：根据预算选择适合的档次

### 7.4 结果操作

计算完成后可执行以下操作：

1. **导出Excel**：将计算结果导出为Excel表格
2. **进入可视化**：查看图表化的分析结果
3. **咨询AI助手**：点击消费档次获取详细建议

- 每条折线代表一种康养模式

## 10. 常见问题

### Q1：为什么计算结果推荐的是这个模式？

A：系统基于余弦相似度算法，将您的兴趣向量与四种模式的标准向量进行计算，匹配度最高的模式即为推荐结果。您可以在可视化分析页面查看详细的对比图表。

### Q2：消费档次的ABCD有什么区别？

A：
- **A档（经济型）**：基础体验，适合预算有限的用户
- **B档（舒适型）**：品质与价格平衡，适合大多数用户
- **C档（品质型）**：高端体验，适合追求品质的用户
- **D档（高端型）**：定制奢华，适合高端需求用户

### Q3：如何导出计算结果？

A：计算完成后，页面会显示"导出Excel"和"进入可视化分析"按钮。点击即可导出数据或查看图表。

### Q4：AI助手无法回答问题怎么办？

A：请检查：
1. 网络连接是否正常
2. DeepSeek API Key是否有效
3. 控制台是否有错误信息

### Q5：系统支持手机访问吗？

A：是的，系统采用响应式设计，完美适配手机、平板、电脑等多种设备。移动端会自动收起侧边导航，点击左上角菜单按钮展开。

### Q6：如何联系开发团队？

A：本系统由石家庄铁道大学开发，如有问题可通过学校联系方式咨询。
